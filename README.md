# ctrl

> A GitHub App built with [Probot](https://github.com/probot/probot) that The CTRL GitHub Bot.

## Setup

```sh
# Install dependencies
npm install

# Run the bot
npm start
```

## Docker

```sh
# 1. Build container
docker build -t ctrl .

# 2. Start container
docker run -e APP_ID=<app-id> -e PRIVATE_KEY=<pem-value> ctrl
```
